### 용어

```
테스트 커버리지
-> 시스템 또는 소프트웨어 테스트의 충분함 정도, 보통 70~80% 목표

MAU
-> 월간 방문 순사용자

피처(feature)
->  서비스의 기능 중심 업무. 예를 들면 버튼을 추가한다는 등의 업무

PR (pull Request)
-> 깃 허브에서 프로젝트에 PR이 반영되었다는 뜻.
-> 자신의 코드가 해당 프로젝트에 기여하였다

컨트리뷰터
-> 오픈소스에 서브 개발자급으로 기여한 사람

메인테이너
-> 오픈 소스에 메인 개발자 급으로 기여한 사람
```
